# Contributing to Glamornate Frontend

This repository is a **public source mirror** of an actively-developed production app. The canonical development happens in a private monorepo. As such:

## What this repo is for
- Code review and reference
- Reporting bugs you observe in the deployed app
- Reading our architecture and runbook documentation

## What this repo is NOT for
- Direct pull requests (we are not currently accepting external PRs)
- Direct dependency consumption (workspace-package sources are vendored; see `src/lib/contracts/` and `src/data/catalog/`)

## How to report issues
File issues via the GitHub issue tracker using the templates in `.github/ISSUE_TEMPLATE/`. We aim to triage within 5 business days.

## Security disclosures
Please **do not** file security issues in the public tracker. See `SECURITY.md` for our private disclosure process.

## Questions
Please use GitHub Discussions or open an issue with the `question` label.
